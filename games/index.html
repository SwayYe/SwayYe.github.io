刀塔霸业 Dota Underlords
PC / MAC / IPHN / IPAD / ANDR / LIN / 策略 / 2019-06-14 / 2019-06-14
2019-08-28 / 自走棋 Steam 不知不觉天亮了 Valve 2019 /
最近玩得比较多的游戏，个人觉得比游廊自走棋做得好。虽然说特效没有游廊酷炫（之后肯定会加的），但是无论是操作手感，还是查看自己、他人羁绊、装备，都比游廊友好太多了。从这里不难看出V社做游戏的水平。另外游戏的物品、英雄从近几个版本也渐渐跟游廊自走棋区分开了，而且个人觉得平衡性和可玩性比游廊自走棋要好。
音乐世界 Cytus II
IPHN / IPAD / ANDR / 音乐/旋律 / 2018-01-18
2019-02-25 / 音乐游戏 手游 Rayark 音乐 剧情 音游 / ★★★★★ 力荐
算是音游入坑作，之前碰过OSU，也玩过节奏大师，但是Cytus II是第一部让我想要把里面所有歌曲都Million Master的游戏，不同人物角色的人设非常有趣，通过IM虚拟社交界面来推动剧情，这些都是在之前玩过的音游中没有碰到过的。雷亚确实是全世界最好的音游制作商，让我不禁想要补上它的之前作品。 通关更新评论：最后的结局还算可以吧，后期为了增加耐玩性，引入了Capso抽奖系统，看别人评论一般，但我觉得还行，我大概从2.5之后的剧情是跟着Capso系统打的，到结局的时候，其实东西基本上已经抽完了，如果从头就有Capso的话，到结局前妥妥能把奖池抽完的。

刺客信条 大革命 Assassin's Creed Unity
2020-07-20 / 刺客信条 育碧 Ubisoft 法国 动作 历史 ACT 角色扮演 / ★★★★☆ 推荐
因为去年巴黎圣母院起火，而免费领到了本体。第一部完整玩下来的刺客信条作品。首先得夸一下育碧在建模方面的用心，光是完整还原圣母院这一块，就足够给他一个高分。可惜的是剧情有些生硬，说实话，我感觉就是每一个序列的任务都差不多，开头过个动画，然后找人、潜入、刺杀。由于第一次玩，我又不太会潜入，所以大部分关卡都是直接狂战士莽过去，战斗这一块体验确实一般。
隐形守护者
2020-03-16 / 互动影视游戏 谍战 剧情 潜伏 交互电影 中国 经典 / ★★★★☆ 推荐
12小时 全成就。 若非生日别人作为礼物送我，我应该不会接触这个国产互动游戏。首先，剧情和演员的表现来说，我是愿意给满分的，可以看出拍摄时的导演，是真的用心了。唯一的一星扣在选项的游戏性上，我后来为了解锁全部的结局，不得不把之前的剧情全部过一遍，选择不同的选项，后来点这些选项点得都快吐了。说一下结局吧，happy ending中，可以说所有人都有了还不错的结局，除了方敏，整部剧里，我认为最惨的就是她了吧，连杀人无数的武藤，最后都能回日本安详晚年，而为了革命牺牲了家人，颠沛流离大半辈子的方敏，最后却以潜伏者的身份牺牲，实在心疼。但转念一想，当时为了信仰牺牲的人，又何止方敏这样一个呢。（P.S. 武藤纯子实在是太可爱了）
GRIS
2020-03-16 / 艺术 独立游戏 唯美 解谜 独立 冒险 单机游戏 Steam / ★★★★★ 力荐
6小时 通关+全成就 与其说是游戏，不如说是艺术品。本身作为横版冒险，没有什么操作难度，解密方面，也非常浅显，只需要静静地享受画面与音乐的完美结合即可。另外小姐姐唱歌是在太好听啦~
死亡细胞 Dead Cells
2020-02-21 / 横版过关 动作 游戏打击感爽，可玩性高 Rogue-Like 像素 冒险 ACT 困难 PC / ★★★★★ 力荐
Steam上第一款全成就的游戏，足以见得我对其的喜爱。流畅的打击感，隐藏在像素背后的细腻画面，有趣的剧情，当然，最重要的，是一次次死亡后，终于攻克难关、击败BOSS的喜悦，以上这些让这款RogLike游戏令人着迷。当你第一次打败国王之手，开始报幕，以为游戏通关的时候，才发现这才刚刚入了门，迎接你的是更棘手的小怪，更难的BOSS，一次次的倒下，又一次次的爬起，不断磨练自己的技术，钻研学习打法，最后手刃收藏夹，身形灰飞烟灭仅剩细胞尚存，看着无法再被拯救的王国，不知为何，竟也有几分空虚伤感。
看门狗 Watch_Dogs
2020-02-02 / 沙盒 Ubisoft 育碧 看门狗 高科技 动作 RPG 犯罪 角色扮演 / ★★★★★ 力荐
我玩完的第一款育碧系RPG大作，这个游戏我初中的时候看别的up主玩，当是觉得骇进整个城市这个设定实在是太酷了，不成想真正玩到是5年之后。这款游戏从趣味性和自由度上，比古墓丽影好太多了，唯一的缺陷恐怕是玩到后来主线任务重复性太高了，无非是去哪儿抢个东西，去哪儿点个灯，打各类小喽啰，一把消音手枪，一把AK，一把榴弹足矣。剧情方面我觉得确实不错，狗哥的人物塑造非常饱满。Uplay上没有计时，我估计是10小时左右通关的，支线没有做。
不动就会死 Move or Die
2020-01-10 / 撕逼 多人 Steam 独立游戏 欢乐 合作 PC 联机 欧美 / ★★★★★ 力荐
无论是本地多人还是在线多人游戏，都十分有趣，本身不动就死的设定就非常新颖，再加上各类突变器，让各类关卡玩起来都变化无穷。
武装原型 Broforce
2020-01-10 / 像素 横版过关 射击 独立游戏 横版动作 美国 2015 动作 / ★★★★★ 力荐
非常优秀的横版冒险游戏，关键是支持多人在线游玩，里面各个角色的技能都非常有趣，跟朋友玩得根本停不下来
中国式家长
2020-01-01 / 中国 模拟经营 人生 独立游戏 模拟 steam 游戏 真实 国产 2018 / ★★★★★ 力荐
我玩过的少有的培养类游戏，让我感触最深的其实是这个游戏过于真实。玩过一遍之后，从第二代开始，为了考上清华，一周日程全用来培养属性和技能点，降压纯靠去商店买东西。为了后代属性值，与女孩子勾心斗角，刷好感度。而如果是女孩子的话，找男孩子刷好感度明显就容易很多，只要安排她跟男孩下课、晚自习待一起就行。其背后的隐喻和讽刺着实入木三分。
死光 Deadlight
2020-01-01 / 横版 僵尸 生存 AVG 动作 解谜 ACT 独立游戏 STEAM / ★★★★☆ 推荐
跑酷类横版冒险游戏。剧情挺老套的，不过画面氛围渲染的很不错。最闹心的其实是跑酷时的各类操作，经常操作不当就GG重来，反而是用武器打僵尸要舒服一些。通关之后还会有一个nightmare（噩梦）模式，必须一命通关，我就没有再玩下去，看了别人的视频，噩梦模式的结局甚是黑暗啊。
古墓丽影：崛起 Rise of the Tomb Raider
2019-08-28 / 冒险 动作 Tomb-Raider 生存 RPG 2015 / ★★★★☆ 推荐
玩完之后，其实没有第一部时的那种感触了。虽然各方面依然很不错，但是想比于第一部，没有什么改变，玩久了之后，难免有些审美疲劳。另外剧情到了第二部，感觉劳拉以及其他人的行为变得有些无法理解。这也是我想比喻第一部，评分变低的理由。
古墓丽影 Tomb Raider
2019-08-28 / 动作 TombRaider SquareEnix 单机游戏 美国 / ★★★★★ 力荐
没有玩过老一代的游戏，直接玩的这个重置版第一部。作为我接触的首个解谜类RPG游戏，体验还是非常好的。动作和战斗系统很顺手，解密元素也非常有意思。剧情也很优秀，让我体会到了劳拉第一次冒险从胆怯无助，到独当一面的成长。可惜我没什么耐性，做完主线剧情之后，就没有再去做支线剧情。
